# NEXIS Website

  Official website for **NEXIS (Next Intelligence Systems)** — AI-Powered Enterprise Software based in Gaborone, Botswana.

  ## Stack
  - React + Vite + TypeScript
  - Tailwind CSS v4
  - Framer Motion
  - Wouter (routing)

  ## Getting Started

  ```bash
  npm install
  npm run dev
  ```

  ## Deploy to Vercel

  Import this repo at [vercel.com/new](https://vercel.com/new). Vercel auto-detects Vite and configures the build.

  ## Pages
  - `/` — Home
  - `/about` — About
  - `/services` — Services
  - `/portfolio` — Work / Portfolio
  - `/contact` — Contact
